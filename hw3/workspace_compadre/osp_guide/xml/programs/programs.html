Demonstration Programs

There is no substitute for a short demonstration program.  This java archive (jar file) contains ready-to-run programs that show how to use the Open Source Physics Code library.  The Open Source Physics: A Users Guide with Examples by W. Christian (Addison-Wesley 2007) provides a description of these programs and the entire Open Source Physics (OSP) project.

Double-clicking on a leaf-node in the Launcher's left-hand "tree" to run a program.  Source code is available on the OSP CD.

  • Chapter 1: Introduction to Open Source Physics
  • Chapter 2: A Tour of Open Source Source Physics
  • Chapter 3: Frames Package
  • Chapter 4: Drawing
  • Chapter 5: Controls and Threads
  • Chapter 6: Plotting
  • Chapter 7: Animation, Images, and Buffering
  • Chapter 8: Visualization of Two-Dimensional Fields.
  • Chapter 9: Differential Equations and Dynamics
  • Chapter 10: Numerics
  • Chapter 11: Three Dimensional Modeling
  • Chapter 12: XML Documents
  • Chapter 13: Video
  • Chapter 14: Utilities

Examples for Chapters 15 through 18 are available in other Launcher packages or from the OSP website:

http://www.opensourcephysics.org.
